# 🦂 Scorpion Cursor

Scorpion Cursor is a creative front-end mini project built entirely using HTML.  
It demonstrates how interactive cursor effects and UI animations can be implemented within a single HTML file using embedded styling and scripting.

## 🚀 Live Demo

🔗 https://dinesh-bavaskar.github.io/scorpion-cursor/
## 🌟 Features

- 🖱️ Custom scorpion-themed cursor
- 🎨 Smooth interaction effects
- ⚡ Lightweight single-file implementation
- 🌐 Deployed using GitHub Pages

## 🛠️ Technology Used

- HTML5

## 🎯 Project Objective

This project showcases creativity and front-end fundamentals by implementing interactive browser behavior without using external frameworks or libraries.
